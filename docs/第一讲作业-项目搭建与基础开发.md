### 3.2 用户注册登录

**登录流程**：

```
用户输入手机号 → 发送验证码 → 验证码存储到Redis(TTL 2min) →
用户输入验证码 → 校验验证码 → 查询/创建用户 → 生成Token →
存储用户信息到Redis(TTL 30min) → 返回Token
```

**核心代码** — `UserServiceImpl.java`：

```java
@Override
public Result login(LoginFormDTO loginForm, HttpSession session) {
    String phone = loginForm.getPhone();
    // 1. 校验手机号
    if (RegexUtils.isPhoneInvalid(phone)) {
        return Result.fail("手机号格式错误");
    }
    // 2. 校验验证码
    String cacheCode = stringRedisTemplate.opsForValue().get(LOGIN_CODE_KEY + phone);
    String code = loginForm.getCode();
    if (cacheCode == null || !cacheCode.equals(code)) {
        return Result.fail("验证码错误");
    }
    // 3. 查询用户
    User user = query().eq("phone", phone).one();
    // 4. 新用户自动注册
    if (user == null) {
        user = createUserWithPhone(phone);
    }
    // 5. 保存用户到Redis
    String token = UUID.randomUUID().toString(true);
    UserDTO userDTO = BeanUtil.copyProperties(user, UserDTO.class);
    Map<String, Object> userMap = BeanUtil.beanToMap(userDTO);
    String tokenKey = LOGIN_USER_KEY + token;
    stringRedisTemplate.opsForHash().putAll(tokenKey, userMap);
    stringRedisTemplate.expire(tokenKey, LOGIN_USER_TTL, TimeUnit.MINUTES);
    // 6. 返回Token
    return Result.ok(token);
}
```

### 3.3 拦截器机制

项目使用**双拦截器**架构实现登录校验与Token自动续期：

| 拦截器 | 作用 | 拦截路径 |
|--------|------|---------|
| `RefreshTokenInterceptor` | 刷新Token有效期 | 所有路径 |
| `LoginInterceptor` | 校验用户是否登录 | 需要登录的接口 |

```java
@Configuration
public class MvcConfigurer implements WebMvcConfigurer {
    @Override
    public void addInterceptors(InterceptorRegistry registry) {
        // Token刷新拦截器 - 拦截所有路径
        registry.addInterceptor(new RefreshTokenInterceptor(stringRedisTemplate))
                .addPathPatterns("/**").order(0);
        // 登录拦截器 - 只拦截需要登录的路径
        registry.addInterceptor(new LoginInterceptor())
                .excludePathPatterns(
                    "/user/code", "/user/login",
                    "/blog/hot", "/shop/**", "/shop-type/**",
                    "/voucher/**", "/upload/**"
                ).order(1);
    }
}
```

### 3.4 商铺查询

`ShopServiceImpl.java` 实现商铺的 CRUD 操作：

```java
@Override
public Result queryById(Long id) {
    // 查询商铺
    Shop shop = getById(id);
    if (shop == null) {
        return Result.fail("商铺不存在");
    }
    return Result.ok(shop);
}

@Override
public Result update(Shop shop) {
    // 先更新数据库
    updateById(shop);
    // 再删除缓存
    stringRedisTemplate.delete(CACHE_SHOP_KEY + shop.getId());
    return Result.ok();
}
```

```javascript
const baseUrl = '/api';
axios.defaults.baseURL = baseUrl;
axios.interceptors.request.use(config => {
    const token = localStorage.getItem('token');
    if (token) {
        config.headers.authorization = token;
    }
    return config;
});
```
