# 系统设计文档

## 1. 项目概述

**项目名称**：生活服务-秒杀商品系统

**项目简介**：本系统是一个分布式生活服务秒杀商品平台，支持用户注册登录、商品浏览、秒杀抢购、订单管理等功能。

**主要功能**：
- 用户模块：用户注册、登录、签到
- 商品模块：商品浏览、商品详情
- 订单模块：订单创建、订单查询
- 库存模块：库存管理、库存扣减

## 7. 用户注册登录功能实现

### 7.1 登录流程
```
用户输入手机号 → 发送验证码 → 验证码存储到Redis → 
用户输入验证码 → 验证验证码 → 查询用户 → 创建Token → 返回Token
```

### 7.2 Redis Key 设计
| Key | 说明 | 过期时间 |
|-----|------|---------|
| login:code:{phone} | 登录验证码 | 5分钟 |
| login:token:{token} | 用户登录Token | 30分钟 |
| user:sign:{userId}:{yyyyMM} | 用户签到记录 | 1个月 |
